# Missing People Custom Theme

A high-impact, professional WordPress theme designed specifically for the **Missing People Reporter** plugin. This theme transforms a standard WordPress installation into a dedicated portal for missing persons investigations, featuring urgent visual cues and deep plugin integration.

## ✨ Key Features
- **Hero Search Portal**: A massive, focused homepage search bar targeting the `missing_person` database.
- **Urgent Visual Identity**: Premium design using a serious Red/Blue/White palette with high-contrast typography (Inter).
- **Deep Plugin Integration**:
    - **Bespoke Profile Grids**: Custom layouts for personal facts, physical descriptions, and circumstances.
    - **Live Map Integration**: Embedded Leaflet maps for last-known locations.
    - **Poster Tool Styling**: Dedicated print-optimized styles for the plugin's "Print Missing Poster" tool.
    - **Risk Alerts**: Dynamic "High Risk" pulsing badges for urgent cases.
- **Automatic Setup**: Programmatically creates the "Report a Case" page with the required shortcode upon theme activation.
- **Fully Responsive**: Mobile-first design for officers and the public in the field.

## 🚀 Installation
1.  Ensure the [Missing People Reporter](https://github.com/Mentaltude/missing-people-reporter) plugin is installed and active.
2.  Download/Clone this repository into your `/wp-content/themes/` directory.
3.  Activate the **Missing People Custom Theme** via the WordPress Dashboard.
4.  Navigate to **Appearance > Menus** to set up your primary navigation.

## 🛠️ Technical Details
- **Base**: Custom-built for performance and accessibility.
- **Typography**: Inter (via Google Fonts).
- **Icons**: Dashicons (WordPress Core).
- **Templates**: Overrides plugin templates (`single-missing_person.php`, `archive-missing_person.php`) for a unified experience.
